Test Instructions

  1. Make sure JustUI extension is enabled and Navigation Guardian is active
  2. Open your browser's DevTools Console (F12 → Console) for detailed logs
  3. Click the JavaScript navigation buttons above
  4. Cross-origin navigations should show confirmation modals
  5. Same-origin navigations should work without modals
  6. Special protocols (mailto, tel, javascript) should be ignored
  7. Check that blocked/allowed counts update in the extension popup
  8. Verify the console output matches expected behavior
