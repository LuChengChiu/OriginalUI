Test Instructions

  1. Make sure JustUI extension is enabled and Navigation Guardian is active
  2. Submit the cross-origin forms above - you should see confirmation modals
  3. Test both "Allow" and "Block" buttons in the modals
  4. Verify that GET and POST methods both work
  5. Check that blocked/allowed counts update in the extension popup
  6. Test forms with target="_blank" open new tabs when confirmed
  7. Verify that same-origin forms work without modals
