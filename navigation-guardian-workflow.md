# Navigation Guardian Workflow

## Overview

The Navigation Guardian is a comprehensive multi-layer protection system that prevents malicious cross-origin navigation attempts, pop-unders, and click hijacking attacks. It operates through four distinct layers working in coordination.

**File**: `src/scripts/injected-script.js`

### Intercepted Methods
- `window.open()` - Blocks pop-unders and malicious pop-ups
- `location.href` assignments - Blocks programmatic redirects
- `location.assign()` and `location.replace()` - Complete navigation coverage
- `addEventListener()` - Blocks malicious click listeners
